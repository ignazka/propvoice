# PropVoice – Nützliche Prompts

## Session starten

```
Lies die CLAUDE.md. Wir sind in Phase [X].
Ich möchte heute [Ziel]. Was ist dein Plan?
```

## Phase abschließen

```
Phase [X] ist fertig. Update die CLAUDE.md:
- Aktuelle Phase auf Phase [X+1] setzen
- Notiere was gebaut wurde (Tabellen, Routen, Komponenten)
```

## Neues Konzept lernen

```
Ich kenne [Konzept] noch nicht. Erkläre mir in 5 Sätzen
wie es funktioniert, bevor wir anfangen.
```

## Debugging

```
Ich habe diesen Fehler: [Fehler]
Erkläre mir zuerst warum er auftritt, dann fix ihn.
```

## Code Review

```
Schau dir [Datei] an. Was würdest du verbessern?
Erkläre jede Änderung kurz.
```

## Supabase RLS

```
Ich brauche eine RLS-Policy für [Tabelle].
Erkläre mir jede Zeile der SQL.
```

## Feststecken

```
Ich verstehe [Thema] nicht. Erkläre es konzeptuell,
dann zeig mir ein konkretes Beispiel aus unserem Projekt.
```
